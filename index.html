Cartographie des réseaux de carrefours

Dans la plupart des régions les sentiers pédestres balisés sont organisés en réseaux de carrefours : tronçons balisés reliant deux carrefours indiqués par des poteaux.

La plupart du temps, la cartographie de ces réseaux est difficile à trouver.

Depuis quelque temps, une équipe de contributeurs intégre l'ensemble de ces réseaux dans la base de données d'OpenStreetMap.

Ces réseaux peuvent déjà être visualisés de plusieurs façons : Waymarkedtrails, Knooppuntnet, Thunderforest outdoors... Le but ici est d'avoir une carte plus personnalisable pouvant être insérée par exemple dans un site d'office de tourisme.
